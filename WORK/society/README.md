## Онтология предметной области
Тема «Я и социум» охватывает взаимодействие подростка с обществом, правовые и финансовые аспекты.  
Выделено 6 крупных подтем, каждая из которых раскрывается через 2–4 конкретных понятия.  
Общее количество понятий – 15.


### Схема связей (онтология)


Связи между понятиями:
- Иерархические: разделы → конкретные темы.
- Горизонтальные:
  - *Трудоустройство* связано с *Мошенничеством при трудоустройстве* и *Официальной работой*.
  - *Права школьника* и *Ответственность* связаны с *Комендантским часом*.
  - *Управление карманными деньгами* перекликается с *Долгами среди друзей*.
  - *Алгоритмы соцсетей* объясняют *Влияние лайков*.
